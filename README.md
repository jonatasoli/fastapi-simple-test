# How run the project?

## Poetry
* Install poetry 
```pip install poetry```
* Run poetry install
```poetry install```
* Enter in venv
```poetry shell```
* Run tests
```pytest```
* Run project
```./start-app.sh```

## Venv
* Create venv 
* Run requirements install
```pip install -r requirements.txt```
* Enter in venv
* Run tests
```pytest```
* Run project
```./start-app.sh```
